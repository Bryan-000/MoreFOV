# MoreFOV

A BepInEx mod for White Knuckle which lets you set your FOV all the way up to 180, and all the down to 0.

## Manual install

* Step 1: Download the mod via clicking this [![download button thing](https://duviz.xyz/static/user/Pc70UdlcOj9-.png)](https://github.com/Bryan-000/MoreFOV/releases/download/1.0.0/MoreFOV.dll "gaayyy :3")
* Step 2: If you don't have BepInEx, get BepInEx [here](https://github.com/BepInEx/BepInEx/releases/download/v5.4.23.5/BepInEx_win_x64_5.4.23.5.zip "you are :3c").
    * Step 2a: Go to your White Knucke install's folder via Steam > Library > right click White Knuckle > Manage > Broswe Local Files
    * Step 2b: Extract the BepInEx zip and dump the BepInEx folder, winhttp.dll, .doorstop_version, and doorstop_config.ini into your White Knuckle install's folder. (next to the White Knuckle.exe)

* Step 3: Go to White Knuckle/BepInEx/plugins (or create that folder if it doesnt exist) and drag in the MoreFOV dll into there.

* Step 4: Run the game :3
